What?

A TitleID is a unique identifier of your Homebrew Application. It defines your app without the need of checking your Title or Description so these don't have to be unqiue (there can be multiple applications with the same name). TitleDB.com tries to prevent conflicts with multiple TitleIDs by maintaining a big database of all Homebrew Applications and their TitleIDs. This website is a small tool to generate a conflict-free TitleID and check if a TitleID is already in use.

Thanks to the help of DanTheMan827, this tool will also check against native 3DS TitleIDs (eShop titles).

What to do next?

You can use the TitleDB for your own Homebrew Application. As soon as you finished development, you should push your CIA or ZIP to titledb.com to be added to the database!
